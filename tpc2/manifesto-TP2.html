Resumo

Este tpc pretendia converter um dataset de JSON para RDF. Para isso procurei um conversor online de JSON para XML.

Tendo o XML passei a definir a ontologia por detrás do dataset, usando o Protégé. Depois usei XSL para converter o dataset em XML para Turtle.

Tendo a ontologia e o dataset em Turtle, juntei ambos para concluir a transformação em RDF.

Finalmente importei o ficheiro final obtido no GraphDB, com sucesso.
