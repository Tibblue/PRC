Resumo

Este tpc pretendia converter um dataset em XML para RDF. Para tal começamos por definir a ontologia por detrás do dataset, usando o Protégé. Depois usamos XSL para converter o dataset em XML para Turtle.

Tendo a ontologia e o dataset em Turtle, juntamos ambos para concluir a transformação em RDF.
