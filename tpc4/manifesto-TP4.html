Resumo

Este tpc pretendia adicionar o conceito de molecula à ontologia da Tabela Periodica.

Foi criada uma classe para a molecula, e depois criei uma object property 'molecula' que pretende relacionar um elemento a uma molecula. Para a relação oposta usa-mos a já existente object property 'elemento' .

Para data properties reusa-mos o 'name' e 'symbol', e criamos duas novas, 'ponto de ebulição' e 'ponto de fusão' que servem exatamente para o que o nome implica.
